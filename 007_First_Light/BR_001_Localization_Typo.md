# Bug Report: Localization Error in Russian Text (Epilepsy Warning Screen)
## Bug ID
BR-001

## Title
Typo in Russian localization — missing letter "р" in word "при" on epilepsy warning screen

## Environment
- Application: 007 First Light
- Platform: PC
- Language: Russian

## Preconditions
- Game is launched with Russian language selected

## Steps to Reproduce
1. Launch the game
2. Observe the epilepsy/health warning screen at startup
3. Read the last bullet point

## Expected Result
"Не играйте **при** ощущении усталости или головной боли."

## Actual Result
"Не играйте **пи** ощущении усталости или головной боли."
